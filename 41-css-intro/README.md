### 1. **Adding CSS**
This section demonstrates three methods of adding CSS to an HTML file:
- **Inline CSS** (directly within HTML tags)
- **Internal CSS** (within a `<style>` tag in the `<head>`)
- **External CSS** (via an external stylesheet)

### 2. **CSS Selectors**
In this section, you will learn how to use various CSS selectors to apply styles:
- **Element selector**: targets elements based on their HTML tag name
- **Class selector**: targets elements based on their class attribute
- **ID selector**: targets elements based on their ID attribute
- **Attribute selector**: targets elements based on attributes and values
- **Universal selector**: targets all elements
