A **desktop typing speed test application** built with **Python and
Tkinter**.\
This project measures **words per minute (WPM)** and **typing accuracy**
while the user types a sample text.

The application includes **Colombian‑inspired text samples** that bring
a bit of the culture, coffee, and energy of cities like **Bogotá,
Medellín, and Cartagena** into the experience.

This project is part of my **100 Python Projects Challenge**, where I
build practical Python applications to strengthen my programming and
software development skills.

# ✨ Features

-   ⌨️ **Typing speed calculation (WPM)**
-   🎯 **Accuracy measurement**
-   ⏱ **Automatic timer starting on first keystroke**
-   🔄 **Restart test functionality**
-   🇨🇴 **Colombian‑inspired typing texts**
-   🖥 **Simple and clean Tkinter GUI**

------------------------------------------------------------------------

# 🧠 How It Works

1.  The program displays a **random sample text**.
2.  The timer starts **when the user begins typing**.
3.  The user types the text in the input box.
4.  When finished, the program calculates:

-   **Words Per Minute (WPM)**
-   **Typing Accuracy (%)**

Average typing speed is around **40 WPM**, but with practice many people
reach **80--100 WPM**.
