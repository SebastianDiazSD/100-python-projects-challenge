## 🌎 Overview

This project presents a comprehensive analysis of global space missions
since the beginning of the **Space Race in 1957**.\
Using a rich dataset from **NextSpaceFlight**, the analysis explores
trends in launches, mission success rates, costs, and the organizations
shaping the modern space industry.

Through data exploration, cleaning, and visualization, the project
transforms historical launch data into clear insights about the
evolution of space exploration.

The work is part of the **100 Python Projects Challenge**, a portfolio
initiative showcasing practical applications of Python for data
analysis.

------------------------------------------------------------------------

## 📊 Key Questions Explored

The analysis investigates several aspects of the global space industry:

-   Which organizations have conducted the largest number of launches?
-   How has the **number of missions evolved over time**?
-   How have **mission costs changed** across decades?
-   Which countries dominate launch activity?
-   Which months tend to have the highest launch frequency?
-   Has the **success rate of space missions improved** over time?

These questions provide a deeper understanding of the transformation
from the **Cold War space race** to today's **commercial space
economy**.

------------------------------------------------------------------------

## 🛰 Dataset

Source: **NextSpaceFlight.com**

The dataset contains historical information about space launches
worldwide, including:

-   Launch date
-   Organization responsible for the mission
-   Launch location
-   Rocket status
-   Mission outcome (success or failure)
-   Estimated mission cost

This data allows for both **historical analysis** and **modern industry
insights**.

## 📈 Analytical Workflow

The project follows a structured data science process:

1.  **Data Loading**\
    Importing and inspecting the raw dataset.

2.  **Data Cleaning**

    -   Date normalization\
    -   Handling missing values\
    -   Converting numeric fields

3.  **Feature Engineering**\
    Extracting temporal variables such as:

    -   Year
    -   Month

4.  **Exploratory Data Analysis**\
    Identifying patterns in:

    -   Launch frequency
    -   Organizational activity
    -   Mission outcomes

5.  **Visualization**\
    Creating clear charts and visual summaries to highlight trends.

------------------------------------------------------------------------

## 📊 Example Insights

Some of the patterns revealed through the analysis include:

-   The **Cold War era** saw rapid growth in launch activity driven
    primarily by the United States and the Soviet Union.
-   In recent years, **commercial space companies** have significantly
    increased launch frequency.
-   Mission reliability has improved considerably over time.
-   Launch activity shows moderate seasonal patterns across months.
