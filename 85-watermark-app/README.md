# 85 - Watermark App

Desktop application built with **Python and Tkinter** that allows users
to automatically add watermarks to images.

Inspired by tools like watermarkly.com.

---

## Features

-   Upload images
-   Add custom text watermark
-   Preview before saving
-   Export watermarked images

---

## Tech Stack

-   Python
-   Tkinter
-   Pillow

---

## Installation

Install dependencies:

    pip install -r requirements.txt

Run the application:

    python main.py

---

## Example Use Case

Perfect for creators that want to automatically add:

-   website
-   logo
-   brand name

to images before posting on social media.

Example watermark:

    sebastiandiaz.dev | Hecho en Colombia 🇨🇴
