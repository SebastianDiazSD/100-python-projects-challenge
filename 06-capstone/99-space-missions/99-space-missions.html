🚀 Project 99 — Space Missions Analysis¶

Part of my 100 Python Projects Challenge.

In this project I explore a dataset containing every space mission since the beginning of the Space Race (1957).

Dataset source:
https://nextspaceflight.com

The dataset includes information such as:

  • Launch organisation
  • Location
  • Mission date
  • Rocket status
  • Mission success/failure
  • Mission price

Goals¶

Using Python, Pandas and Data Visualization, we will answer questions such as:

  • Who launched the most missions each year?
  • Which organisations dominate the space industry?
  • How has mission cost changed over time?
  • Which months have the most launches?
  • Are space missions becoming safer?

This project focuses on:

  • Data exploration
  • Data cleaning
  • Data visualisation
  • Extracting insights from real-world data

Data Cleaning¶

We notice:

  • Two unnecessary columns (Unnamed)
  • Date column is stored as text
  • Price contains missing values

Let's clean the dataset.

In [7]:
df = df.drop(columns=["Unnamed: 0", "Unnamed: 0.1"])
In [8]:
# Convert Date
df["Date"] = df["Date"].str.replace(r"^\w+\s", "", regex=True)
df["Date"] = pd.to_datetime(df["Date"], format="mixed", errors="coerce",utc=True)

# Extract useful time columns
df["Year"] = df["Date"].dt.year
df["Month"] = df["Date"].dt.month

#Inspect missing values
df.isna().sum()

# Price column cleanup
df["Price"] = pd.to_numeric(df["Price"], errors="coerce")

Key Insights¶

1. Space race dominance¶

Early launches were dominated by USA and USSR.

2. Rise of private companies¶

Companies like SpaceX are responsible for the modern increase in launch frequency.

3. Mission success rate¶

Space missions have become significantly safer over time.

4. Costs¶

Mission costs vary widely depending on rocket type and organisation.

5. Launch timing¶

Certain months see slightly higher launch frequencies, likely due to weather and orbital conditions.
