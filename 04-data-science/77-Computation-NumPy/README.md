## 🚀 What This Challenge Covers

In this project, I explored:

- Creating arrays manually with `np.array()`
- Generating arrays using `.arange()`, `.random()`, and `.linspace()`
- Analyzing the shape and dimensions of an `ndarray`
- Slicing and subsetting arrays using indices
- Performing linear algebra operations (scalars & matrix multiplication)
- Using NumPy broadcasting to make shapes compatible
- Manipulating images as NumPy arrays

All explained clearly and structured cleanly.
