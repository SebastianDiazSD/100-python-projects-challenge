# 📊 Data Exploration – College Major Salaries  
### Project 72 of 100 Python Challenges 🇨🇴

This project is part of my **100 Python Projects Challenge**.

In this notebook, I explore salary data across different college majors using **Pandas** inside **Jupyter Notebook (Anaconda environment)**.

The objective is to analyze:
- Starting median salaries
- Mid-career salaries
- Salary risk (spread between 10th and 90th percentiles)
- Salary differences across academic groups (STEM, Business, HASS)

## 🔎 Key Analysis Performed

✔ Data cleaning using `.dropna()`  
✔ Identifying highest and lowest salaries  
✔ Finding majors with highest long-term earning potential  
✔ Calculating salary spread (risk measurement)  
✔ Sorting values using `.sort_values()`  
✔ Grouping data using `.groupby()` (similar to Excel Pivot Tables)

---

## 📈 Key Findings

- **Chemical Engineering** shows the highest mid-career median salary.
- **Economics and Finance** dominate the highest 90th percentile salaries.
- **Spanish** and **Education** have the lowest salary averages.
- STEM majors show the highest average earnings overall.
- Some majors present high earning potential but also high income variability (higher risk).
