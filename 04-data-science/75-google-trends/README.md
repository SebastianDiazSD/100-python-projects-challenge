## 🟡 Overview

In this project, I explore how Google search behavior reflects
real-world events.

Using Google Trends data, we analyze how search volume relates to:

-   🪙 Bitcoin price movements\
-   🚗 Tesla stock performance\
-   📉 The U.S. unemployment rate

The goal is to understand whether collective online behavior can provide
insights into economic and financial trends.

Because sometimes, before markets move... people Google.

------------------------------------------------------------------------

## 🛠 Skills Practiced

This project focuses on:

-   Time-series data analysis
-   Resampling data (daily → monthly)
-   Handling missing values and NaNs
-   Data cleaning with pandas
-   Advanced Matplotlib visualization techniques
-   Dual-axis plotting
-   Date formatting using locators
-   Identifying seasonality patterns

## 📂 Dataset Sources

The project uses:

-   Tesla Search Trend vs Price dataset
-   Bitcoin Search Trend dataset
-   Daily Bitcoin Price dataset
-   U.S. Unemployment Benefits Search vs Unemployment Rate dataset

All datasets are included in the repository.

------------------------------------------------------------------------

## 📈 Key Insights

### 🚗 Tesla

Search spikes often align with large stock price movements.\
Public attention and volatility appear closely related.

### 🪙 Bitcoin

Search interest frequently increases near price peaks.\
This suggests retail interest intensifies during hype cycles.

### 📉 Unemployment

Searches for unemployment benefits rise alongside increases in
unemployment rate --- showing a strong real-world behavioral
correlation.
