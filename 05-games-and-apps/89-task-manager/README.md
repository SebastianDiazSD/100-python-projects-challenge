**Project 89 --- 100 Python Projects Challenge**

Cafe Productivity Board is a simple **task management web application**
built with Flask.\
The project helps users organize their work using a **Kanban-style
board** with three stages:

-   To Do
-   In Progress
-   Done

The idea behind this project is inspired by the productive atmosphere of
**Colombian cafés 🇨🇴**, where a good cup of coffee often helps people
focus and turn ideas into action.

------------------------------------------------------------------------

# 🌍 Features

-   Create new tasks
-   Move tasks between workflow stages
-   Delete tasks
-   Visual Kanban-style task board
-   Simple and clean interface
-   SQLite database for task storage

The application helps developers or students keep track of their work in
a clear and organized way.

# 🚀 Running the Project

Clone the repository

    git clone https://github.com/SebastianDiazSD/100-python-projects-challenge.git

Navigate to the project folder

    cd 89-task-manager

Install dependencies

    pip install -r requirements.txt

Run the application

    python app.py

Open your browser and go to

    http://127.0.0.1:5000
