A fun **Tic‑Tac‑Toe desktop game built with Python and Tkinter** as part
of my **100 Python Challenges**.

This version includes a **Colombian 🇨🇴 themed interface**, while keeping
the gameplay and language fully in **English**.

------------------------------------------------------------------------

## 🎮 Features

-   🧑‍🤝‍🧑 **Player vs Player mode**
-   🤖 **Player vs AI mode**
-   🇨🇴 **Colombian inspired colors (yellow, blue, red)**
-   🪟 **Tkinter graphical interface**
-   🔄 **Reset / New Game button**
-   🧠 **Simple AI strategy**
-   🎯 **Win detection (rows, columns, diagonals)**

## 🚀 Installation

Clone the repository:

``` bash
git clone https://github.com/SebastianDiazSD/100-python-projects-challenge.git
cd 100-python-projects-challenge/84-tic-tac-toe
```

Run the game:

``` bash
python game.py
```

------------------------------------------------------------------------

## 🧠 How the AI Works

The AI follows a simple strategy:

1.  Win if possible
2.  Block the opponent if they are about to win
3.  Take the center square
4.  Take a corner
5.  Choose any available square
