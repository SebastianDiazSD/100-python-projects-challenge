A **desktop writing application** built in Python that challenges writers to keep typing without stopping.

Inspired by **The Most Dangerous Writing App**, this project deletes everything you've written if you stop typing for more than **5 seconds**.

To make the experience more creative, the app includes **Colombian-inspired writing prompts**, bringing a touch of storytelling from places like **Bogotá, Medellín, Cartagena, and the Amazon rainforest**.

Think of it like writing in a small café in Bogotá ☕ — if you stop, the story disappears.

---

# 🚀 Features

- ⌨️ Real-time writing editor
- ⏱ Automatic inactivity detection
- 💥 Text deletion if the user stops typing
- 🇨🇴 Random Colombian-themed writing prompts
- 🖥 Clean and simple desktop interface

---

# 🧠 Project Objective

Writer’s block is a common challenge for many writers.  
This project forces **continuous writing** by removing the possibility of overthinking.

The rule is simple:

> **Keep writing or lose everything.**
