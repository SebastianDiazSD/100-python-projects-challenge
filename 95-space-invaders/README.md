------------------------------------------------------------------------

# 🎮 Game Overview

The player controls a defensive spacecraft positioned at the bottom of
the screen.\
A fleet of alien ships advances toward Earth. The objective is simple:

**Defend the planet by eliminating the invading fleet before they reach
the player.**

Alien units move horizontally across the screen and descend gradually
with every cycle.\
If any alien reaches the player's position, the game ends.

Strategically placed **defensive barriers** provide temporary protection
from enemy advancement.

# ▶️ Running the Game

Ensure Python **3.x** is installed.

Run the game from the project directory:

The game window will open automatically.
