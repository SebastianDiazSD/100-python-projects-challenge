A **command-line Morse Code converter** built as part of my **100 Python
Projects Challenge**.

This program converts:

-   Plain text ➜ Morse code
-   Morse code ➜ Plain text

The project focuses on practicing **Python fundamentals, modular
programming, input validation, and working with dictionaries and JSON
files.**

------------------------------------------------------------------------

# 🚀 Features

-   Encode **text to Morse code**
-   Decode **Morse code to readable text**
-   Simple **command-line interface**
-   Morse dictionary stored in **JSON**
-   **Input validation** with clear error messages
-   Clean **modular Python structure**

# ⚙️ Installation

Clone the repository:

``` bash
git clone https://github.com/SebastianDiazSD/100-python-projects-challenge.git
cd 100-python-projects-challenge/82-morse-encoder
```

Install dependencies:

``` bash
pip install requests beautifulsoup4
```

------------------------------------------------------------------------

# ▶️ Run the Program

From inside the `82-morse-encoder` folder run:

``` bash
python main.py
```

You will see a menu like this:

    0 → Help
    1 → Plain Text → Morse Code
    2 → Morse Code → Plain Text
    9 → Exit
