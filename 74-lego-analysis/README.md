## 📊 Project Objective

The goal of this project was to:

- Merge LEGO datasets
- Analyze historical trends
- Visualize growth patterns
- Understand how LEGO evolved over time

---

## 🧱 Datasets Used

- `sets.csv`
- `themes.csv`
- `colors.csv`

The datasets include information about:

- Set names
- Release year
- Number of parts
- Themes
- Colors
- Transparency

---

## 📈 Key Insights

### 1️⃣ LEGO Growth Over Time
The number of sets released per year has grown significantly, especially after the 1990s.

### 2️⃣ Increasing Complexity
The average number of pieces per set has increased dramatically.  
Modern sets are far more complex than early releases.

### 3️⃣ Expansion of Themes
LEGO diversified into multiple storytelling universes:
- City
- Castle
- Space
- Star Wars
- Marvel
- Architecture

Innovation and adaptability explain LEGO’s long-term success.
